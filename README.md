# 🌐 CyberNode: Invasão de Rede

O **CyberNode** é um jogo estratégico baseado na Web com uma temática puramente cyberpunk e de hacking. O objetivo do jogador é invadir e dominar uma infraestrutura de rede simulada, navegando por nós (nodes) e gerenciando recursos para quebrar as defesas do sistema.

---

## 🎓 Contexto Acadêmico

Este projeto foi desenvolvido como trabalho prático para a disciplina de **Matemática Discreta II** do curso de **Sistemas de Informação** no **Instituto Federal da Bahia (IFBA)**.

O objetivo principal do software é aplicar os conceitos teóricos de **Teoria dos Grafos** vistos em sala de aula dentro de uma simulação interativa e gamificada.

### 📐 Aplicação Prática dos Conceitos:
* **Vértices e Arestas:** A infraestrutura de rede infectada é modelada como um grafo, onde cada servidor/máquina é um **vértice (nó)** e os caminhos de conexão física ou lógica entre eles são as **arestas**.
* **Caminhamento e Adjacência:** Para avançar no jogo, o jogador só pode se mover para nós adjacentes (vizinhos conectados), simulando a propagação real de um vírus ou invasão em sistemas de redes interligadas.
* **Algoritmos de Busca:** A lógica de geração e validação dos caminhos da rede utiliza os princípios de busca e conectividade em grafos para garantir que o mapa seja sempre desafiador e solucionável.
